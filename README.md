# ViewHunt Pro 🚀
YouTube Viral Trend Discovery & AI Strategist

ViewHunt Pro is an advanced intelligence suite for content creators, designed to identify real creation opportunities using local algorithms and AI-powered strategy.

## Core Features
- **Market Trends**: Deep regional analysis (Tanzania, Nigeria, Kenya, Global) to identify Viral, Fast-Growing, and Underserved (New Creators) niche opportunities.
- **Viral Scraper**: Keyword-based discovery for historical viral patterns.
- **AI Video Ideas**: Intelligence-driven concept generation for high-retention video strategies.
- **Channel Auditor**: In-depth performance audit with AI strategic planning.
- **Video Intelligence**: Deep AI-powered analysis of any specific YouTube video link.
- **Idea Notepad**: Central hub for banking and organizing your next viral moves.

## Tech Stack
- **Frontend**: React (Vite), Vanilla CSS, Lucide Icons
- **Backend**: Node.js, Express, Axios
- **Database**: Supabase
- **AI**: LLaMA-3 (via Pollinations API)
- **Data Source**: YouTube Data API v3

## Getting Started
1. Clone the repository.
2. Install dependencies: `npm install`
3. Configure your `.env` file with `YOUTUBE_API_KEY`, `SUPABASE_URL`, and `SUPABASE_SERVICE_ROLE_KEY`.
4. Run the development server: `npm run dev`
5. Start the backend data engine: `node server.js`
